ABC Tags to CSV Extractor

Extracts all ABC Header tags to a CSV file

Instructions:  

1) Click "Open One or Multiple .ABC Files for Tag Extraction" to open one or multiple files containing ABC tunes.

2) Select the ABC files you want to extract the tags from.

3) Click "Open".

4) When requested, enter a filename for the exported CSV file.

5) Click "Save".

The ABC tag columns in the saved CSV are:

Filename,X:,T:,Subtitles,C:,O:,R:,L:,M:,Q:,K:,D:,S:,Z:,N:,r:,ShareURL
