ABC Transcription Tools Website-to-ABC Extraction Tool

Instructions:

This tool can extract all of the ABC tunes from a interactive website exported by the ABC Transcription Tools.

Tunes will not be extracted from websites tunebooks created with "Disable access to editor" option selected.

1) Click "Open a website .html file previously exported from the ABC Transcription Tools" to open a .html file.

The website code will be analyzed and any extracted ABC tunes will appear in the "Extracted ABC Tunes" box.

2) Save, Copy or Test the results:

Click "Save Output to a file" to save the results to a file.

Click "Copy Output to the clipboard" to copy the results to the system clipboard for pasting into other applications.

Click "Test Output in the ABC Transcription Tools" to open and test the results in my ABC Transcription Tools.
